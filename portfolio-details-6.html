Project Description

Developed a mixed reality-based application for real-time simulation of air traffic in 3D. Applied UX design process with air traffic controllers for the design and development of application according to the user needs and practices followed in air traffic management. Developed the application from scratch based on the usersâ€™ feedback about the user interface, interaction design, and usability. According to the user study, which included user interviews and usability testing, users experienced a better understanding of air traffic scenarios through a holographic mixed reality interface compared to 2D screens. Moreover, air traffic controllers considered the application helpful for bad weather conditions.
Skills: C#, Unity3D, Hololens, User-centered design, User research
